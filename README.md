# CHAY_CE Site

## Run locally

```bash
npm install
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Included downloads

- `public/downloads/ElvuiSave.zip`
- `public/downloads/ChayBar.zip`

## Not included in this lite ZIP

`ChayImages.zip` is too large for reliable ChatGPT download packaging. Add it manually here:

```txt
public/downloads/ChayImages.zip
```

Then update `app/page.tsx` to enable the ChayImages button.
